◆ MRCC_voidInit()

void MRCC_voidInit ( void  )

This function is responsible of Initializing Reset Clock Control.

Parameters
none
Returns
none

This function initialize the system clock:

  1. Check that the system clock is External or PLL or Internal then wait for the Ready flag then switch to to chosen Clock
  2. if The chosen Clock is external ,Function will check if it's source is Crystal or RC then Check if Clock Security Enabl or Disable
  3. if The chosen Clock is PLL ,Function will check if PLL's source is HSI or HSE then set PLL Multiplication Factor and PLL Prescaler
  4. Wait for the ready flag in Chosen Clock then Switch to chosen clock
